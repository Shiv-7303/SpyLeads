# SpyLeads Backend

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
2. Copy `.env.example` to `.env` and fill in your secrets.
3. Initialize the database:
   ```bash
   python init_db.py
   ```
4. Run the Flask app:
   ```bash
   flask run
   ```

## Testing

Run tests with:

```bash
pytest
```
